Überblick

Newton-Spiegel-Rechner

Berechnet Kontrast- und Schärfeverluste sphärischer Hauptspiegel in Newton-Teleskopen gegenüber dem idealen Parabolspiegel. Kombiniert Wellenfront-Optik (Strehl, MTF) mit einem wahrnehmungsphysiologischen Modell (CSF) und stellt alle Ergebnisse interaktiv in sechs Diagramm-Tabs dar.

Abhängigkeiten: Python ≥ 3.10 · matplotlib · numpy · tkinter (Stdlib)
Start: python newton_spiegel_rechner.py

2 · Physikalische Grundlagen

Wellenfront, Strehl, MTF, Q_vis

2.1 Wellenfrontfehler

Ein sphärischer Spiegel erzeugt sphärische Aberration 4. Ordnung. Der Wellenfrontfehler hängt von Öffnung D, Brennweite f und Wellenlänge λ ab.

W_PtV (paraxial) Wp = D⁴ / (1024 · f³ · λ) [in Einheiten λ]
W_PtV (best focus) Wb = Wp / 4
W_RMS Wrms = Wb / (1.5 · √5)
Öffnungsverhältnis N = f / D

Best Focus: Durch Defokussierung (Balancierung der 4. Ordnung mit einem Defokus-Term) wird der RMS-Wellenfrontfehler minimiert. Wellenfront: W(ρ) = 8·Wb·(ρ⁴ − ρ²)

2.2 Strehl-Quotient

Misst das Peak-Intensitätsverhältnis gegenüber dem beugungsbegrenzten Parabolspiegel.

Maréchal-NäherungS ≈ exp(−(2π · Wrms)²) [gültig für Wrms < 0.07λ]
Exakter Strehl S = |∫₀¹ exp(i·2π·W(ρ)) · 2ρ dρ|² [Pupillenintegral]
W(ρ) best focus W(ρ) = 8·Wb·(ρ⁴ − ρ²)
Eff. Öffnung D_eff = D · S^0.25 (Kontrast)
Hinweis: Die Maréchal-Näherung überschätzt den Strehl bei schnellen Spiegeln (f/D < 8) deutlich. Ab v5.0.1 wird für alle wahrnehmungsrelevanten Berechnungen der exakte Strehl aus dem Pupillenintegral verwendet.

2.3 MTF

Die MTF beschreibt die Kontrastübertragung als Funktion der normierten Ortsfrequenz ν ∈ [0, 1], wobei ν = 1 der Beugungsgrenzfrequenz fc = D/(λ · 206265) [arcsec⁻¹] entspricht.

MTF Paraboloid mp(ν) = (2/π)·(arccos(ν) − ν·√(1−ν²))
MTF Sphäre (rel.)msr(ν,Wb) = |∫ exp(i·2π·ΔW(x)) dx| / Normierung
MTF Sphäre (abs.)msa(ν,Wb) = msr(ν,Wb) · mp(ν)
Grenzfrequenz fc = D / (λ[mm] · 206265) [arcsec⁻¹]

Zwei Fokus-Modi:

  • Best Focus (Standard): W(x) = 8·Wb·(x⁴ − x²) — entspricht Zemax / telescope-optics.net
  • Paraxialer Fokus: W(x) = 8·Wb·x⁴ — entspricht Sacek / Born & Wolf

2.4 Visueller Qualitätsindex Q_vis

Q_vis misst die wahrgenommene Bildqualität über das CSF-gewichtete MTF-Integral.

Q_vis (exakt) ∫₀¹ msr·mp·CSF(ν·fc·3600/V) dν ÷ ∫₀¹ mp·CSF(ν·fc·3600/V) dν
CSF(f) 2.6·(0.0192 + 0.114·f)·exp(−(0.114·f)^1.1) [van Meeteren]
Q̃ Näherung S + (1−S)·(1 − 1/(1+(V_krit/V)²))
V_krit D · 0.7 · √S_exakt [nach Suiter]
Gültiger V-Bereich: 30–400×. Bei sehr kleinen V dominiert das Auge (CSF ≈ 0 für hohe Frequenzen → Q ≈ 1), bei sehr großen V entsteht ein numerisches Artefakt durch den CSF-Peak. Die Näherungsformel Q̃ ist im mittleren Bereich um typisch 1–3 % zu optimistisch.

2.5 Wb-Inversion (Bisektionssuche)

Frühere Versionen invertierten die Maréchal-Näherung und überschätzten Wb um Faktor ≈ 2. Ab v5.0.2 wird korrekt bisektiert:

def _Wb_von_S(S, n=60):
    # findet Wb sodass _strehl_exakt(Wb) == S
    lo, hi = 0.0, 0.44
    for _ in range(n):
        mid = (lo + hi) / 2.0
        if _strehl_exakt(mid) > S: lo = mid
        else:                     hi = mid
    return (lo + hi) / 2.0

7 · Bekannte Einschränkungen

Modellgrenzen

  • Nur sphärische Aberration 4. Ordnung. Koma, Astigmatismus und höhere Ordnungen werden nicht berücksichtigt.
  • CSF nach van Meeteren: mittlere Näherung. Individuelle Augenunterschiede und Adaptationszustand nicht modelliert.
  • Q_vis ist nur im Bereich V = 30–400× physikalisch sinnvoll.
  • Q̃-Näherung überschätzt die Qualität bei mittleren V typisch um 1–3 %. Die exakte Kurve ist das zuverlässigere Modell.
  • _strehl_exakt() ist nicht LRU-gecacht. Bei kurven_N() mit vielen Stützstellen kann die Berechnung einige Sekunden dauern.
